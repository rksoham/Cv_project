PROJECT TITLE:
Real-Time Morphology-Based Object Counting System

DESCRIPTION:
This project detects and counts objects in real time using
morphological image processing techniques and displays the
results on a web interface.

TECHNOLOGIES USED:
- Python
- OpenCV
- Flask
- NumPy
- HTML/CSS

HOW TO RUN:
1. Install dependencies using requirements.txt
2. Run: python app.py
3. Open browser and go to http://127.0.0.1:5000

APPLICATIONS:
- Industrial object counting
- Traffic monitoring
- Inventory management
- Agricultural produce counting
